# 연구자 사용 가이드 (USAGE)

이 저장소는 두 산출물을 제공합니다.

1. **의료취약지 리포트/지도** (공개) — `index.html`(리포트) + `map.html`(지도).
   상급종합병원 접근성·고령화·검진 수검률·질환부담을 결합한 분석. Vercel 배포.
2. **합성 건강검진 데이터 생성기** (인증) — `synthetic/`.
   KOSIS 검진통계 집계 분포로부터 가상 개인 검진 레코드를 생성. 구글 로그인 + 승인 이메일.

## 합성데이터 사용 시 유의

- **가상 데이터입니다.** KOSIS 집계 분포에서 난수로 생성한 합성 레코드로, 실존 인물과
  1:1 대응이 없습니다. 실측 마이크로데이터가 아니며, 임상 의사결정 근거로 쓸 수 없습니다.
- **파생값**: LDL은 Friedewald 식, eGFR은 CKD-EPI 2021로 산출한 결정값입니다(원자료 분포와
  의도적으로 다를 수 있음, 검증 탭에 "파생값" 표기).
- **시군구 모드**: 인구구조(연령·성)는 시군구, 검사수치 분포는 소속 시도 기준입니다
  (KOSIS가 검사수치를 시도·연령까지만 제공). 시군구별 수검률 차이는 반영되지 않습니다.
- **재현·인용**: 생성 후 **데이터카드(JSON)**를 함께 내려받으세요. 시드·상관강도·연도·출처·
  파라미터가 기록되어, 동일 조건+동일 시드로 완전히 동일한 데이터를 재현할 수 있습니다.

### 권장 인용 문구(예)

> 본 연구는 KOSIS 국민건강보험공단 건강검진통계(<연도>)의 집계 분포로부터 가우시안 코퓰라·
> 분위수 매핑으로 생성한 합성 건강검진 데이터를 사용하였다(생성 조건은 동봉 데이터카드 참조).
> 합성 데이터는 실측 개인정보가 아니며 가명·익명정보로 간주된다.

## 재식별 위험

생성기의 "재식별 위험성평가" 탭에서 k-익명성(기준 k≥3, k≥5 충족률 병기)·l-다양성·
t-근접성(민감속성=종합판정)을 제공합니다. 합성 데이터라 직접 재식별 위험은 구조적으로
낮으나, 1세 단위·소표본은 희소도가 올라가니 연령대 일반화를 권장합니다.
